# WithView

## 📢 서비스 소개

캠을 자유롭게 움직일 수 있는 반응형 화상채팅 서비스

## 🤔 기획 배경

시공간의 제약으로 만나기 힘든 20,30대들을 위해 실제로는 떨어져 있지만, 만나서 노는 것처럼 색다른 추억을 만들어주고 싶었습니다.

## 💻 주요 기능

1. 화상 채팅 서비스 (캠 위치 이동, 화면 공유)
2. 서버 생성 및 초대, 즐겨찾기
3. 친구 추가, 메시지
4. 서버 배경 꾸미기 (배경 사진 변경 및 스티커 추가)
5. 현재 배경화면을 프리셋으로 저장
6. 창작마당에서 프리셋 공유 및 저장
7. 회원 정보 관리
